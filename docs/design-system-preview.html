BRAINROT SYSTEM

Roster screams, chrome whispers. A foundational system for a TFT-style Roblox autobattler with an Italian-brainrot character roster. Saturated primaries, sticker outlines, one flourish per component, reward maximalism earned — not sprayed.

Do / Don't

Do

  • Pair every rarity color with a shape glyph.
  • Use one flourish per component.
  • Reserve reward-burst for earned moments.
  • Keep LuckiestGuy for display / stingers only.
  • Test text at UIScale 1.4 before shipping.
  • Put Accessibility as a first-class Settings tab.

Don't

  • Rely on red vs. blue alone for team or rarity.
  • Stack bounce + glow + sparkle on one button.
  • Fire reward-burst on hover or routine clicks.
  • Use LuckiestGuy or Impact for body copy.
  • Default to "…" truncation for overflow.
  • Bury motion-reduction under Advanced.
