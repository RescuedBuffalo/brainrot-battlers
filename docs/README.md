# Docs

Design and planning artifacts for the Brainrot Battlers autobattler.

## Contents

- **`design-system.md`** — v0.1 design system spec. Covers tokens (color, typography, spacing, motion), components (buttons, panels, unit cards, shop slots, HUDs, toasts, tooltips), screen patterns for match / meta / onboarding / social surfaces, accessibility targets, and voice & copy rules. Source of truth.
- **`design-system-preview.html`** — self-contained visual preview. Open in a browser to see every token, component, and screen mock rendered together. Useful for quick review and for sanity-checking token values.
- **`tokens.json`** — DTCG / Figma Tokens Studio format export of the full token set. Import into Figma via the Tokens Studio plugin: File → Import → Tokens Studio JSON.

## Direction

Aesthetic: **"chaotic-brainrot readable game UI"** — the roster is loud (memes in character design, VO, lore, reward moments), the chrome is tight (clean HUDs, legible tooltips, uncluttered shops). Not full maximalist chaos, not polished premium.
